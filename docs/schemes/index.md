# Scientist Schemes

Scientist schemes are reusable agent orchestration patterns. They do not own task-specific metrics or run history; evaluation cells do.

| Scheme | Status | Best Use |
| --- | --- | --- |
| [AutoResearch](autoresearch.md) | active | Fixed metric-driven experiment loops. |
| [AutoScientist](autoscientist.md) | active | Parallel research teams with critique and synthesis. |
