# Preferences

## Agent Shape

- System style: Codex-native AI Lab with one reusable AI scientist loop.
- Primary use: hard-task research orchestration.
- Autonomy: semi-autonomous.
- Memory home: local-first.
- Routine source scope: web plus connected Drive, Gmail, Calendar, and Slack when relevant.
- Default research output: concise brief plus source map.
- Automatic writes: local files only.
- Run observability: W&B/Weave is required for normal AI scientist runs.
- Connector writes: approval required every time.
- Privacy boundary: no secrets ingestion.
