# Scheme Guide: autoscientist

Date: 2026-06-10
Status: active

## Purpose

Use this scheme when a task benefits from multiple parallel attempts, critique, and synthesis rather than a single metric loop.

## Operating Model

An evaluation cell using this scheme should define shared state, clear work-unit ownership, proposal review rules, and a synthesis cadence. The scheme should make comparisons easier by keeping each method attempt and critique as separate evidence.
