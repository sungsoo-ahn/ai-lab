# Scheme Guide: autoresearch

Date: 2026-06-10
Status: active

## Purpose

Use this scheme when the task has a measurable target, reproducible commands, and a useful cycle of proposal, execution, scoring, and synthesis.

## Operating Model

An evaluation cell using this scheme should define a fixed run spec, clear source gates, artifact paths, and a synthesis prompt. Work units should be narrow enough to preserve failed trials and compare against a local baseline.
