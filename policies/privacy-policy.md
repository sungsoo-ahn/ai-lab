# Privacy Policy

## No Secrets Ingestion

Never store raw credentials, API keys, tokens, passwords, private keys, recovery codes, session cookies, or equivalent secrets in:

- research briefs;
- source maps;
- local memory;
- logs;
- connector drafts or copied notes.

## Redaction

If a source contains a secret, redact the value and note only that a secret exists when that fact is necessary. Use a pointer to the source location instead of copying the value.

## Sensitive Personal Data

Minimize personal data from Gmail, Calendar, Drive, and Slack. Store only what is needed for the research goal, and prefer high-level summaries over raw transcripts.
